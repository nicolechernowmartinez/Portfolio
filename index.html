Agentics Growth Lab

Digital Banking AI Operations Assistant

The Problem

SMB banking teams handle high volumes of exploratory customer questions that consume contact center time without requiring financial advice or account access. These questions — about products, rates, and options — are informational by nature, but they're routed through the same channels as complex service requests, creating bottlenecks and driving up cost per interaction.

The Solution

A friendly AI assistant designed as a first-pass digital banking experience — explaining financial options in plain language without giving regulated advice, collecting personal data, or replacing human bankers. The assistant handles the exploratory conversation, creates clarity for the customer, and hands off to a human with full context when appropriate.

  • Explains financial products and options in plain language
  • Never gives regulated financial advice or collects personal data
  • Designed to complement, not replace, the human banking relationship
  • Reduces contact center volume for non-advice exploratory calls

Customer Journey

Step 1 — Customer calls bank with a product question
Step 2 — Greeted and asked: "Do you have questions about our products?"
Step 3 — Call routed to AI first-pass assistant
Step 4 — AI engages in friendly, clear conversation and explains options
Step 5 — "What would you like to do next?" — warm handoff to human banker

Finance of America

Five9 Call Center Integration for Loan Origination Platform

Executive Summary

Designed and delivered a real-time integration between Five9's cloud call-center platform and a home-grown loan origination system — transforming voice calls from isolated support interactions into fully connected financial workflow events. Every call became part of the loan record, compliance trail, and operational reporting structure.

The Before State

  • Agents worked in Five9 while loan officers worked in the LOS — two separate systems
  • Call data lived entirely separately from loan application data
  • Context switching between platforms slowed resolution times
  • Compliance evidence was fragmented across systems
  • Managers lacked end-to-end visibility into borrower interactions

Product Vision

Create a voice-enabled financial workflow platform where every call is a connected event:

  • Calls automatically attach to the correct loan application
  • Agents operate inside a single unified interface
  • Call metadata becomes part of the permanent financial record
  • Compliance evidence is generated by default, not manually assembled

Key Initiatives

1. Embedded Call Panel in the Loan Platform

Built a native call management interface directly inside the LOS — incoming call pop-ups showing borrower and loan context, one-click call controls, live loan details visible during conversation, and inline notes stored directly to the loan record.

Impact: Eliminated system switching for agents. Agents never needed to leave the loan platform.

2. Automatic Call-to-Loan Linking

Matching logic using phone number normalization and active application lookup. Every call produced a structured record: Loan ID, Borrower ID, Agent ID, Timestamp, Call type, and Outcome — automatically.

Impact: Complete interaction history per loan, built without manual agent data entry.

3. Compliance and Audit Trail Engine

Stored structured call metadata alongside recording references, disposition codes, agent actions, and notes in a queryable format designed for regulatory review.

Impact: Simplified regulatory audits and reduced compliance team effort per review cycle.

4. Agent Productivity Dashboard

Built reporting visibility across calls per loan stage, conversion by call outcome, handle time vs. funding success rate, missed call rates, and queue performance by team.

Impact: Enabled data-driven staffing decisions and coaching conversations grounded in real operational data.

5. Event-Driven Architecture

Built on Five9 webhooks, REST APIs, and asynchronous event processing with idempotent updates — ensuring no duplicate records and full fault tolerance under high call volumes.

Impact: Reliable, scalable integration that held up under peak origination periods without engineering intervention.

Results

  • Significant reduction in average call handling time
  • Higher loan application completion rates tied to better agent context
  • Improved agent satisfaction and retention
  • Faster underwriting cycle times through connected workflows
  • Stronger compliance posture with auto-generated audit trails
  • Improved executive visibility into call-driven revenue
